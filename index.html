Kali Linux

Di bawah ini saya akan menjelaskan pengertian, sejarah, kelebihan dan kekurangan, jenis-jenis, serta cara menginstall Kali Linux di VirtualBox.

Pengertian Kali Linux

Kali Linux, sebuah sistem operasi (OS) open-source, dirancang untuk keperluan hacking dan pengujian penetrasi pada jaringan komputer. Dirilis pada 2013 oleh Offensive Security, Kali Linux merupakan turunan dari Debian Linux dan menjadi standar industri untuk pengujian penetrasi dan forensik digital. Dengan lebih dari 600 alat hacking dan pentesting seperti nmap, metasploit, dan aircrack-ng, Kali Linux memiliki fokus pada keamanan dan privasi serta dapat digunakan sebagai sistem operasi utama atau sebagai OS live pada USB atau CD.

Sejarah Kali Linux

Pertama kali diluncurkan pada 2013, Kali Linux menggantikan BackTrack Linux, distribusi Linux yang telah lama digunakan dalam bidang keamanan informasi sejak 2006. Offensive Security merilis Kali Linux dengan fitur-fitur lebih lengkap dan dukungan yang lebih baik, menjadikannya pilihan utama para profesional keamanan informasi. Sejak itu, Kali Linux telah menjadi distribusi Linux populer untuk kegiatan hacking dan penetration testing.

Kekurangan dan kelebihan Kali Linux

Kekurangan:

1. Tidak cocok untuk pengguna pemula: Dirancang khusus untuk ahli keamanan, mungkin sulit bagi pemula.

2. Kurangnya dukungan resmi: Tidak didukung secara resmi oleh vendor tertentu.

3. Kurang stabil: Sering mendapatkan pembaruan, bisa kurang stabil dan mengalami bug.

Kelebihan:

1. Terintegrasi dengan berbagai alat keamanan: Lebih dari 600 alat keamanan terintegrasi, memudahkan tes penetrasi.

2. Gratis dan Open Source: Tersedia secara gratis dan open source, tanpa biaya lisensi.

3. Kompatibilitas hardware yang baik: Dapat diinstal pada berbagai jenis hardware dan kompatibel dengan perangkat keras.

4. Komunitas yang besar: Mendukung oleh komunitas pengguna besar dan aktif.

Jenis-jenis Kali OS

1. Kali Linux Full: Mengandung lebih dari 600 alat hacking untuk berbagai jenis tes penetrasi.

2. Kali Linux Light: Cocok untuk perangkat keras lebih tua atau pengguna yang membutuhkan OS dasar.

3. Kali Linux ARM: Dukung ponsel dan tablet, termasuk Raspberry Pi dan Odroid.

4. Kali Linux Virtual: Versi untuk diinstal sebagai mesin virtual di atas OS utama.

5. Kali Linux NetHunter: Fokus pada pengujian penetrasi perangkat seluler Android dan iOS.

5. Kali Linux Docker: Versi yang berjalan di dalam wadah Docker.

5. Kali Linux AWS: Instal pada layanan cloud Amazon Web Services (AWS).

Cara menginstall Kali Linux di virtualBox

Di bawah ini adalah panduan langkah demi langkah untuk menginstall Kali Linux di VirtualBox. Pastikan Anda sudah mengunduh VirtualBox dan ISO Kali Linux terlebih dahulu.

1. Buka aplikasi VirtualBox yang telah Anda install.

2. Create Virtual Machine dengan klik tombol New > isi nama, sistem operasi, dan folder.

3. Memory size. Pilih ukuran RAM yang akan dialokasikan untuk Virtual Machine Kali. Pada contoh kami memakai 1GB. Lalu pilih Next.

4. Hard Disk. Pilih sesuai default, yaitu Create virtual had disk now. Lalu klik tombol Create.

5. Hard Disk file type. Pilih default VDI. Lalu klik tombol Next.

6. Storage on physical hard disk. Pilih default Dynamically allocated. Lalu klik tombol Next.

7. File allocated and size. Menentukan jumlah hard disk yang akan digunakan untuk Virtual Machine ini. pada contoh kami mengisi 10GB. Lalu klik tombol Create.

8. Virtual Machine Kali linux sudah jadi dengan status Powered Off.

Setelah Virtual Machine selesai dibuat dengan langkah-langkah diatas. Langkah selanjutnya kita akan setting storage untuk memilih file ISO Kali dan mengatur processor.

9. Pilih tombol Setting > Storage > empty > pilih file iso Kali yang sudah di download, seperti gambar dibawah ini.

10. Selanjutnya ke menu System untuk mengatur Processor virtual machine. Pada contoh ini kami memakai 2 core. Lalu klik OK

Pada langkah selanjutnya kita akan coba menjalankan virtual machine Kali linux yang sudah dikonfigurasi diatas.

11. Klik START. Jika muncul pop up start up disk, pilih Kali Linux yang sudah kita pilih sebelumnya kemudian klik start.

12. Setelah itu pilih Graphical Install, supaya proses instalasinya lebih mudah.

13. Ikuti langkah-langkah instalasi, mulai dari pilih bahasa English > select your location (pilih other – Asia – Indonesia). Klik Continue.

14. Pilih kombinasi bahasa (United States) > keyboard layout (American English). Klik Continue, tunggu proses selesai.

15. Setelah proses selesai kita akan melanjutkan konfigurasi hostname pada machine. Gunakan satu kata sebagai indentitas machine.

Pada contoh kami memakai hostname “kalilinuxrumahweb”. Klik tombol Continue.

16. Pada langkah pengisian domain name, bisa kita lewati. Klik Continue.

17. Setup full name, username dan password, kami menggunakan name: “Kali Rumahweb”, user “kalirumahweb”, dan password “*******”. Continue lalu tunggu proses selesai.

18. Setelah itu pilh zona waktu yang akan digunakan, pada contoh kami menggunakan WIB. Klik tombol Continue.

19. Pada pengaturan partisi disk, pilih default Guide – use entri disk > lalu pilih ATA VBOX HARDDISK > pilih All in one partition. Kemudian Continue.

20. Otomatis akan dibuatkan dua partisi, primary sebagai file system dan logical sebagai swap. Pilih Finish Partitioning and write to disk. Klik Continue

21. Pada tahap ini pilih “Yes”, untuk konfirmasi pengisian konfigurasi sebelumnya. Lalu Continue.

Tunggu sampai proses file sistem linux selesai, karena akan membutuhkan waktu beberapa menit.

22. etelah selesai maka akan muncul pilihan Software yang akan diinstall. Pada contoh ini kami memilih default > lalu Continue.

Instalasi software pada langkah ini memerlukan waktu paling singkat 1 jam. Tergantung konfigurasi RAM dan Processor yang kita gunakan di Virtual Machine.

Semakin besar RAM dan Processor yang kita gunakan, maka semakin cepat instalasinya. Tunggu sampai proses instalasi selesai.

23. Setelah instalasi software selesai, selanjutnya klik Yes untuk instalasi GRUB boot loader. Lalu pilih /dev/sda, agar tidak mengisi secara manual. Klik Continue.

24. Finish Instalasi. Klik continue. Selanjutnya VM akan melakukan restart dan tunggu sampai selesai.

Selamat! Anda Telah Berhasil Menginstal Kali Linux di VirtualBox
